\# 🚚 FedEx Global Supply Chain Optimization



\## 📌 Project Overview

End-to-end Exploratory Data Analysis and Machine Learning 

project analyzing FedEx's global pharmaceutical supply chain 

dataset with 10,324 shipment records across 33 variables.

\## 📊 Project Structure

1\. Data Loading \& Exploration

2\. Data Wrangling \& Cleaning

3\. Exploratory Data Analysis (15 Charts)

4\. Hypothesis Testing (3 Hypotheses)

5\. Feature Engineering \& Preprocessing

6\. ML Model Implementation

&#x20;  - Linear Regression

&#x20;  - Random Forest

&#x20;  - XGBoost ⭐ (Best Model)

7\. Model Deployment (Joblib)

8\. Conclusion



\## 📈 Key Findings

\- Air is dominant shipment mode (6,113 shipments)

\- South Africa is top destination (1,406 shipments)

\- Weight and Freight Cost strongly correlated (0.74)

\- Cameroon has highest average freight cost ($32,006)

\- XGBoost achieved best prediction performance
